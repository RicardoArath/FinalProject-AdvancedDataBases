1.- Abrir la instancia
2.- Subir el archivo zip
3.- Descomprimir el archivo zip con el siguiente comando unzip archivo.zip
4.- Ubicarse en la carpeta del archivo "HospitalElRegio" con cd HospitalElRegio
5.- Realizar el comando EXPORT FLASK RUN APP=app.py
6.- Despues run flask --host=0.0.0.0
7.- Ubicarte en tu navegador y buscar <tuIpExterna>:5000
8.- Podras DISFRUTAR de la experiencia de "HospitalElRegio"!!!
